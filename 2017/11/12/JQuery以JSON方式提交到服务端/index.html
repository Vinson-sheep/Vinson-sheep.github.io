最近的项目需要给服务端提交JSON，JSON本来应该是很简单的，但是我还是折腾了好一段时间。博主我后端使用的是nodejs。

前端

一般来说我比较喜欢用$.ajax来提交数据：

值得注意的是，前端代码即使声明了dataType:"json"，也需要将数据通过JSON.stringify()转换为json字符串再进行传输。JQuery不会你的对象转换为JSON格式，data传输始终都是一串有规律的字符串，数据类型的声明只是附加的内容，用来告诉服务端这是JSON数据。
