单例设计模式——让类创建的对象，在系统中只有唯一的一个实例。每一次执行 类名() 返回的对象，内存地址是相同的

__new__ 方法

  • 使用 类名() 创建对象时，Python 的解释器 首先 会 调用 __new__ 方法为对象 分配空间
    __new__ 是一个 由 object 基类提供的 内置的静态方法，主要作用有两个：
    • 1) 在内存中为对象 分配空间
    • 2) 返回 对象的引用
  • Python 的解释器获得对象的 引用* 后，将引用作为 第一个参数**，传递给 __init__方法
