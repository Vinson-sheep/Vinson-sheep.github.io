记录python类属性和类方法，并且引申静态方法

类属性

  • 类属性 就是给 类对象 中定义的 属性
  • 通常用来记录 与这个类相关 的特征
  • 类属性 不会用于记录 具体对象的特征

类方法

  • 类方法 就是针对 类对象 定义的方法
  • 在 类方法 内部可以直接访问 类属性 或者调用其他的 类方法

    • 也 不需要 访问 类属性 或者调用 类方法
